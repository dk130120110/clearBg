clearBg
=======

起因是图片去白底，面向百度编程，但由于英语太菜，能找到的只有一键去白的效果，然而我想要的是绕开角色自身的白色，只清除背景的白色。

去白底不能清除闭合的区间，可用涂抹清理细节的部分。看效果图大概就明白了。

由于本人编程水平有限，或有更好的实现方法，以上。
